API Documentation

Temperature

Get the temperature in celsius or fahrenheit
Supports single letter varients because spelling celsius and fahrenheit is hard

Request

GET /temperature/c
        GET /temperature/celsius
        GET /temperature/f
        GET /temperature/fahrenheit

Query Parameters

city required
string

state
string

country required
string

Response
